# 捐助说明

OPanel是一个非盈利性质的开源项目，但出于项目开发和维护的需要，我们开放捐助入口。

你可以在<https://nocp.space/donate>对本项目进行捐助，下面是对不同捐助情况的详细说明。

## 个人

### 任何金额

如果你已加入[OPanel交流群](https://qm.qq.com/q/1t1VHv1eHa)，且在捐助时附上了你的QQ号信息，那么可以在群内获得“赞助者”头衔。

:::info

如果是之前捐助过本项目，但没有获得群头衔的，可以私信群主获得群头衔。

:::

### ≥ ¥30

可以在[友链](/links/)页面添加一个友情链接。

## 云服务商

### 任何金额 / 合作

如果你已经以组织代表人的身份加入[OPanel交流群](https://qm.qq.com/q/1t1VHv1eHa)，且在捐助时附上了你的QQ号信息，那么可以在群内获得“赞助商”头衔。

### 合作

如果你们与OPanel达成了合作关系，那么可以在[友链](/links/)页面添加一个友情链接。
