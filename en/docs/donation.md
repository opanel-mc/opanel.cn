# Donation

OPanel is a non-profit open-source project. To support ongoing development and maintenance, we welcome donations.

You can make donation at <https://nocp.space/donate>. Details for different donation scenarios are below.

## Individuals

### Any amount

If you have joined the [OPanel QQ group](https://qm.qq.com/q/1t1VHv1eHa) and include your QQ id with your donation, you can receive the "赞助者" title in the group.

:::info

If you previously donated but did not receive the group title, you can DM the group owner to obtain it.

:::

### ≥ ¥30

You can add a friend link on the [Links](/en/links/) page.

## Cloud Server Providers

### Any amount / Partnership

If you have joined the [OPanel QQ group](https://qm.qq.com/q/1t1VHv1eHa) as an organization representative and include your QQ id with your donation, you can receive the "赞助商" title in the group.

### Partnership

If you establish a partnership with OPanel, you can add a friend link on the [Links](/en/links/) page.
